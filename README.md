# Rock

This is a simple multicore operating system that is made for fun

# Features

- x86_64 kernel
  - PMM
  - VMM
  - SMP
  - APIC
  - preemptive multicore scheduler
  - ext2 FS
- Basic drivers
  - keyboard (ps2)
  - mouse (ps2)
  - ahci
  - pci
- GUI
  - cursor
  - clickable widgets 
  - wallpaper
  - panel
